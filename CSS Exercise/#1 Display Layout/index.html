Display Block

  • Display block, Lebar defaultnya selebar browser
  • defaultnya akan menambahkan garis baru
  • bebas mengatur tinggi dan lebar
  • di dalamnya dapat menyimpan banyak elemen seperti inline -block, inline, atau bahkan block lagi

Display Inline

  • Tidak menambahkan garis baru
  • Lebar dan Tinggi sesuai konten yang ada
  • tidak dapat mengatur tinggi nyaa
  • YAP SEPERTI INI

Display Inline-Block

  • Perilakunya Sama Sepert inline tetapi ini bisa mengatur tinggi dan lebarnya
  • harus secara manual menambahkan elemen ini
  • Tidak ada elemen yang punya default inline-block
