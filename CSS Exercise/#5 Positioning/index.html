Static position

  • Value default untuk semua elemen jika tidak di beri property position

Relative Position

  • Elemen akan naik 1 dimensi
  • Ruang sebelumnya masi ada

Absoulte Position

  • Elemen naik 1 dimensi, dan ruang elemen absolute ini menghilang
  • Elemen absolute akan bergerak relatif terhadap parent (Parent non-static)
  • *(class absolute ini ber property absolute)
  • Element naik 1 dimensi
  • Bergerak relatif terhadap window
  • *(biasanya value ini digunakan untuk membuat navbar atau top bar dengan CSS)
